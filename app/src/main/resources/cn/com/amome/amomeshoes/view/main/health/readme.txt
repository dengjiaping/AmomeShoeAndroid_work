1、HealthFragment：健康模块页面。对应布局文件fragment_health_main.xml。
该页面包含四个分页,依次为ServiceFragment（日常自诊）、DetectionFragment（智能检测）、ReportFragment（健康报告）、PromotionFragment（健康促进），点击界面按钮可以进入对应页面。
2、service包含日常自诊相关的页面。日常自诊包含量一量、看一看、我的鞋、爱心助手四个功能。
3、detection包含智能检测相关的页面。智能检测包含站一站、蹲一蹲、摇一摇、走一走四个功能。
4、report包含健康报告相关的页面。健康报告包含足部报告、姿态报告、平衡报告、步态报告。
5、MarketWebviewActivity为商城页面，由HealthFragment进入。
