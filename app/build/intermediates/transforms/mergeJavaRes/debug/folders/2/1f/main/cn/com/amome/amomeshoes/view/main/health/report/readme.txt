1、ReportFragment：健康报告页面。对应的布局文件fragment_report_main.xml。该页面包含四个详细报告。（智能检测最后生成的是简略报告）
2、FootReportActivity：足部报告页面（走一走测试生成）。分为压力图谱、足形评估、足部报告几个方面。
3、PostureReportActivity：姿态报告页面（站一站测试生成）。分为压力图谱、压力分布、姿态评估、姿态报告几个方面。
4、BalanceReportActivity：平衡报告页面（摇一摇测试生成）。分为重心轨迹、评估指标、整体均衡性评估、平衡报告几个方面。
5、WalkingReportActivity：步态报告页面（走一走测试生成）。分为步伐/步幅、单/双支撑相、步态状况几个方面。
