1、WalkDetectionActivity 点击走一走进入该页面。如果没有连接智能鞋会先连接智能鞋并获取电量，电量不足会弹出相应提示是否继续，最后进入准备界面；如果已连接，会直接进入准备界面。在准备界面，点击查看教程会进入教程页面（WalkTeachingActivity），点击开始测试会播放注意事项语音，播放结束后正式进入测试。测试过程中会收集上报的64点数据。在测试完成后，会将这些原始数据进行运算，具体算法详见（util-CalDetection）（如果用户未进行行走生成错误数据会给出相应提示）计算完成后将数据推至服务端，并跳转至简略报告界面，展示用户步态情况和双脚对称性分析并给出建议（来自服务端）。
2、WalkTeachingActivity 摇一摇教程页面。显示操作步骤和规范。点击开始测试会回到上一页面开始测试。
3、WalkReportActivity 摇一摇简略报告页面。展示用户步态情况和双脚对称性分析并给出建议（来自服务端）。点击查看完整报告会进入步态报告（WalkingReportActivity）。
