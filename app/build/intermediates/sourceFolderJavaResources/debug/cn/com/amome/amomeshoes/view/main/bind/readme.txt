1、BindActivity 绑定功能的主页面。点击进入扫描二维码页面（SweepScanActivity），扫码完成跳转到震动测试页面（BindShockActivity），连接智能鞋并开始震动测试，震动测试完成跳转到压力测试页面（BindStressActivity），压力测试完成跳转到计步测试页面（BindWalkActivity），计步测试完成展示绑定结果（成功/失败）。
2、SweepScanActivity 扫描二维码页面。
3、BindShockActivity 震动测试页面。
4、BindStressActivity 压力测试页面。
5、BindWalkActivity 计步测试页面。
6、PrepareScanActivity 准备绑定页面，如果在使用智能检测模块时，未绑定智能鞋，会弹出此页面。
