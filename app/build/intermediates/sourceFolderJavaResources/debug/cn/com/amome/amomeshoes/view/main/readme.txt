1、MainFragmentActivity：登录成功后的主页面。对应布局文件：fragment_ui_main.xml。
主界面包含四个分页，依次为健康（HealthFragment）、运动（ExerciseFragment）、活动（ActivityFragment）以及我（MyFragment），点击界面按钮可以进入对应页面。
2、bind包含绑定模块相关的页面。
3、health包含健康模块相关的页面。健康模块包含日常自诊、智能检测、健康报告、健康促进。
4、exercise包含运动模块相关的页面。运动模块包括饼状图分析、评估、雷达图分析、历史数据。
5、activity包含活动模块相关的页面。
6、my：包含“我”模块相关的页面。
