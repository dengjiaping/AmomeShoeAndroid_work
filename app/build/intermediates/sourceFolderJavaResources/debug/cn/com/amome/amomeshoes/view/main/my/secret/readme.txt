1、FootSecretActivity 脚的秘密页面。展示用户看看脚，量量脚，蹲一蹲中的测试结果。基本情况中显示的脚长，脚宽，脚肤色，脚型，脚印，如果未设置，默认图片为灰色，设置后为绿色，足弓和前后旋如果没有问题，不显示，有问题显示相应的图片。脚痛/脚疾类别中会显示所设置的脚痛/脚疾，如果未设置默认为空，设置了为绿色图片，并且可以点击查看相应病症的原因和建议，这些信息是从服务端请求。
2、ShoeSecretActivity 鞋的秘密页面。是用户我的鞋中添加的鞋信息的汇总。最观察中展示鞋总数，最钟爱中展示鞋种类前三的名称，最信赖中展示鞋品牌前三的名称，最百变中展示鞋跟高前三的名称，最吐槽中展示鞋问题前三的名称。涉及到排名的，服务端返回数据，客户端进行排序。
